# Data Acquisition

## Equipment Used

- **UAV**: DJI Air 2S drone platform
- **GPS Equipment**: Trimble GeoXH (2008 series) with external Zephyr 2 antenna
- **Mobile LiDAR**: 
  - iPhone LiDAR scanner
  - GeoSlam system with Trimble Catalyst DA2 antenna
- **Photography**: iPhone camera system
